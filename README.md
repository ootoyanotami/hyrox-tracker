# HYROX Training Tracker - PWAアプリ

## 📱 スマホにインストールする方法

### iPhone (Safari)
1. Safariで `hyrox-tracker.html` を開く
2. 画面下部の「共有」ボタン（□に↑のアイコン）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ
5. ホーム画面にアプリアイコンが追加されます！

### Android (Chrome)
1. Chromeで `hyrox-tracker.html` を開く
2. 右上のメニュー（︙）をタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択
4. 「追加」または「インストール」をタップ
5. ホーム画面にアプリアイコンが追加されます！

## 🚀 必要なファイル
以下のファイルを同じフォルダに配置してください：
- `hyrox-tracker.html` - メインのアプリファイル
- `manifest.json` - アプリ設定ファイル
- `service-worker.js` - オフライン対応用
- `icon-192.png` - アプリアイコン（小）
- `icon-512.png` - アプリアイコン（大）

## 🌐 Webサーバーで公開する方法

### 方法1: 無料ホスティングサービス（おすすめ）

#### Netlify
1. https://www.netlify.com/ にアクセス
2. 「Sign up」で無料登録
3. 全ファイルをフォルダにまとめてZIP化
4. サイトにドラッグ&ドロップ
5. 自動的にURLが発行されます！

#### Vercel
1. https://vercel.com/ にアクセス
2. GitHubアカウントでログイン
3. ファイルをアップロード
4. デプロイ完了！

#### GitHub Pages
1. GitHubにリポジトリを作成
2. 全ファイルをアップロード
3. Settings > Pages で公開設定
4. `https://username.github.io/repo-name/` でアクセス可能

### 方法2: ローカルサーバー（テスト用）
```bash
# Python 3がインストールされている場合
cd /path/to/hyrox-tracker
python3 -m http.server 8000

# ブラウザで http://localhost:8000/hyrox-tracker.html を開く
```

## ✨ PWA の機能
- ✅ ホーム画面から直接起動
- ✅ アプリのようなフルスクリーン表示
- ✅ オフラインでも動作
- ✅ データはブラウザに保存（デバイス間で同期されません）

## 📝 注意事項
- PWAとして動作させるには、HTTPSまたはlocalhostでの実行が必要です
- データはブラウザのローカルストレージに保存されます
- ブラウザのデータを削除すると記録も消えます
- 定期的にバックアップを取ることをおすすめします

## 🔧 トラブルシューティング
- アイコンが表示されない → キャッシュをクリアして再読み込み
- インストールできない → HTTPSで公開されているか確認
- データが消えた → ブラウザの設定でサイトデータが許可されているか確認
